Making My first website

This was my first website I had built. It was relatively simple as you can see. When designing the navigation for the site, I used jQuery instead of Bootstrap. I added some jQuery to make a smooth scroll effect that made it a bit easier for mobile users to control as well.

Implementing Responsive design

Even though this was my first website, I made sure to make it responsive because I knew that about 80% of people view websites and apps on their phones. Using media queries I made the site fluid and designed it so that it could adapt to any device out there. Now back then I didn't know about Bootstrap but I still made it work with CSS.

The navigation changes when on mobile and it shows the links vertically. Since I was farely knew to web development I thought it would be a great idea to create something that would show what I knew.

You can find the source code on my Github along with several other projects.
