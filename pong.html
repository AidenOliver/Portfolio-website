Pong

Made with HTML and vanilla Javascript. I built this game to help me better understand the Javascript syntax.

Pong.js

One of my earlier projects, pong. To get a grasp of the JavaScript syntax I used Object Oriented Programming. This can be very helpful for when there is a lot of JavaScript code, this helps the developer understand the code a lot easier by assigning objects to variables and functions. This is seen a lot within game development, but also with web development too.

I used pong because of the simplicity of the game and because it helped me undertand the core principles of Object Oriented Programming. Even though the game has a relatively easy concept, it was still very challenging to make.
